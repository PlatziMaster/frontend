# frontend

## React Curriculum vitae

Challenge de curriculum hecho en react para participar del programa de Ethereum Developer

### Instalación

```
npm install
```

### Ejecución

```
npm run start
```

### Server

```
npm run server
```

### Compilar

```
npm run build
```

### Pruebas Unitarias

```
npm run test
```

### ESlint

```
npm run lint
```
