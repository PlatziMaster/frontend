# frontend

Projecto desarrollado como reto de Platzi Master.

Proyecto desarrollado desde 0 con wepack, code en React.js, estilos en Stylus y pruebas unitarias en Jest y Enzyme

### Instalación

```
npm install
```

### Ejecución

```
npm run start
```

### Server

```
npm run server
```

### Compilar

```
npm run build
```

### Pruebas Unitarias

```
npm run test
```

### ESlint

```
npm run lint
```
